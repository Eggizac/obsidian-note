# Определения и формулировки теорем по дифференциальным уравнениям

## 1. Основные понятия

**Определение 1.1.** Обыкновенное дифференциальное уравнение (ОДУ) — соотношение вида:
$$F(x, y, y', y'', \ldots, y^{(n)}) = 0$$

**Определение 1.2.** Порядок ДУ — это порядок его старшей производной.

**Определение 1.3.** Решение ДУ — функция $y = y(x)$, которая при подстановке обращает уравнение в тождество.

**Определение 1.4.** Общее решение — семейство решений, зависящее от $n$ произвольных постоянных для уравнения порядка $n$.

**Определение 1.5.** Задача Коши — задача нахождения решения ДУ, удовлетворяющего начальным условиям $y(x_0) = y_0, y'(x_0) = y_1, \ldots, y^{(n-1)}(x_0) = y_{n-1}$.

**Определение 1.6.** Интегральная кривая — график решения ДУ на плоскости $(x, y)$.

**Определение 1.7.** Поле направлений уравнения $y' = f(x, y)$ — совокупность направлений (с угловым коэффициентом $f(x, y)$), определённых в каждой точке области.

**Определение 1.8.** Условие Липшица по переменной $y$: существует константа $L > 0$ такая, что
$$|f(x, y_1) - f(x, y_2)| \leq L|y_1 - y_2|$$

---

## 2. Теоремы существования и единственности

**Теорема 2.1 (Коши-Пикара).** Пусть функция $f(x, y)$ и её частная производная $\dfrac{\partial f}{\partial y}$ непрерывны в прямоугольнике $\Pi = \{(x, y) : |x - x_0| \leq a, |y - y_0| \leq b\}$. 

Тогда существует единственное решение задачи Коши, определённое в окрестности $|x - x_0| \leq h$, где $h = \min\{a, \tfrac{b}{M}\}$ и $M = \max_{(x,y) \in \Pi} |f(x, y)|$.

**Теорема 2.2 (Существования).** Если $f(x, y)$ непрерывна в прямоугольнике $\Pi$, то существует хотя бы одно решение задачи Коши в окрестности $|x - x_0| \leq h$.

## 3. ДУ первого порядка

**Определение 3.1.** ДУ с разделяющимися переменными имеет вид $y' = f(x)g(y)$ или $M_1(x)N_1(y)dx + M_2(x)N_2(y)dy = 0$.

**Определение 3.2.** Однородное ДУ первого порядка имеет вид $y' = f(y/x)$.

**Определение 3.3.** Уравнение в полных дифференциалах: $M(x, y)dx + N(x, y)dy = 0$, где $\dfrac{\partial M}{\partial y} = \dfrac{\partial N}{\partial x}$.

**Определение 3.4.** Линейное ДУ первого порядка имеет вид $y' + p(x)y = q(x)$.

**Определение 3.5.** Уравнение Бернулли имеет вид $y' + p(x)y = q(x)y^{\alpha}$, где $\alpha \neq 0, 1$.

**Определение 3.6.** Особое решение — решение, в каждой точке которого нарушается условие единственности задачи Коши.

**Определение 3.7.** Огибающая семейства кривых $\Phi(x, y, C) = 0$ — кривая, касающаяся каждой кривой семейства.

---

## 4. Линейные ДУ высшего порядка

**Определение 4.1.** Линейное однородное ДУ $n$-го порядка (ЛОДУ):
$$y^{(n)} + a_1(x)y^{(n-1)} + \cdots + a_n(x)y = 0$$

**Определение 4.2.** Линейное неоднородное ДУ $n$-го порядка (ЛНДУ):
$$y^{(n)} + a_1(x)y^{(n-1)} + \cdots + a_n(x)y = f(x), \quad f(x) \not\equiv 0$$

**Определение 4.3.** Функции $y_1(x), \ldots, y_n(x)$ линейно зависимы на $(a, b)$, если существуют константы $C_1, \ldots, C_n$ (не все нулевые) такие, что
$$C_1y_1(x) + \cdots + C_ny_n(x) \equiv 0$$

**Определение 4.4.** Функции $y_1(x), \ldots, y_n(x)$ линейно независимы на $(a, b)$, если из равенства выше следует $C_1 = \cdots = C_n = 0$.

**Определение 4.5.** Определитель Вронского (вронскиан) функций $y_1, \ldots, y_n$:
$$W(x) = \begin{vmatrix}
y_1 & y_2 & \cdots & y_n \\
y_1' & y_2' & \cdots & y_n' \\
\vdots & \vdots & \ddots & \vdots \\
y_1^{(n-1)} & y_2^{(n-1)} & \cdots & y_n^{(n-1)}
\end{vmatrix}$$

**Теорема 4.1 (Об эквивалентности).** Для решений ЛОДУ $n$-го порядка следующие утверждения эквивалентны:
1. Функции $y_1, \ldots, y_n$ линейно зависимы на $(a, b)$
2. $W(x) \equiv 0$ на $(a, b)$
3. Существует точка $x_0 \in (a, b)$ такая, что $W(x_0) = 0$

**Определение 4.6.** Фундаментальная система решений (ФСР) ЛОДУ $n$-го порядка — система из $n$ линейно независимых решений.

**Теорема 4.2 (Об общем решении ЛОДУ).** Если $y_1(x), \ldots, y_n(x)$ — ФСР ЛОДУ $n$-го порядка, то общее решение:
$$y(x) = C_1y_1(x) + \cdots + C_ny_n(x)$$

**Теорема 4.3 (Лиувилля-Остроградского).** Для решений ЛОДУ $n$-го порядка выполняется:
$$W(x) = W(x_0)\exp\left(-\int_{x_0}^x a_1(s)ds\right)$$

**Теорема 4.4 (О структуре решения ЛНДУ).** Общее решение ЛНДУ имеет вид:
$$y(x) = y_{\text{одн}}(x) + y_{\text{ч.р.}}(x)$$
где $y_{\text{одн}}$ — общее решение соответствующего однородного уравнения, $y_{\text{ч.р.}}$ — частное решение неоднородного.

## 5. ДУ с постоянными коэффициентами

**Определение 5.1.** Характеристическое уравнение для ЛОДУ с постоянными коэффициентами:
$$y^{(n)} + a_1y^{(n-1)} + \cdots + a_ny = 0$$
имеет вид:
$$\lambda^n + a_1\lambda^{n-1} + \cdots + a_n = 0$$

**Теорема 5.1 (Случай 1: различные вещественные корни).** Если $\lambda_1, \ldots, \lambda_n$ — различные вещественные корни характеристического уравнения, то ФСР:
$$y_k(x) = e^{\lambda_k x}, \quad k = 1, \ldots, n$$

**Теорема 5.2 (Случай 2: вещественный корень кратности $m$).** Если $\lambda_0$ — вещественный корень кратности $m$, то соответствующие $m$ решений:
$$e^{\lambda_0 x}, \quad xe^{\lambda_0 x}, \quad x^2e^{\lambda_0 x}, \quad \ldots, \quad x^{m-1}e^{\lambda_0 x}$$

Вклад в общее решение:
$$y_{\lambda_0}(x) = (C_1 + C_2x + \cdots + C_mx^{m-1})e^{\lambda_0 x}$$

**Теорема 5.3 (Случай 3: комплексные корни).** Если $\lambda = a \pm ib$ ($b \neq 0$) — пара комплексных корней, то вещественные решения:
$$y_1(x) = e^{ax}\cos(bx), \quad y_2(x) = e^{ax}\sin(bx)$$

Вклад в общее решение:
$$y_{a \pm ib}(x) = e^{ax}(C_1\cos(bx) + C_2\sin(bx))$$

## 6. Метод неопределённых коэффициентов

**Определение 6.1.** Специальная правая часть — функция из классов: многочлены, экспоненты $e^{ax}$, тригонометрические функции и их произведения.

**Теорема 6.1 (Выбор формы частного решения).**

| Правая часть | Форма $y_{\text{ч.р.}}$ | Множитель $x^s$ |
|---|---|---|
| $P_m(x)$ | $x^s(A_0 + A_1x + \cdots + A_mx^m)$ | если 0 — корень кратности $s$ |
| $e^{ax}$ | $x^sCe^{ax}$ | если $a$ — корень кратности $s$ |
| $e^{ax}P_m(x)$ | $x^se^{ax}(A_0 + \cdots + A_mx^m)$ | если $a$ — корень кратности $s$ |
| $e^{ax}(P_m\cos bx + Q_m\sin bx)$ | $x^se^{ax}(\tilde{P}\cos bx + \tilde{Q}\sin bx)$ | если $a \pm ib$ — корень кратности $s$ |

## 7. Механические колебания

**Определение 7.1.** Уравнение механических колебаний:
$$m\ddot{x} + \beta\dot{x} + kx = F(t)$$
где $m > 0$ — масса, $\beta \geq 0$ — коэффициент затухания, $k > 0$ — жёсткость пружины.

**Определение 7.2.** Свободные колебания: $F(t) \equiv 0$; вынужденные колебания: $F(t) \not\equiv 0$.

**Теорема 7.1 (Классификация свободных колебаний).** Обозначим $\gamma = \tfrac{\beta}{2m}$, $\omega_0 = \sqrt{\tfrac{k}{m}}$.

При $\beta = 0$: гармонические колебания с частотой $\omega_0$.

При $0 < \beta < 2\sqrt{mk}$ (т.е. $\gamma < \omega_0$): затухающие колебания.

При $\beta = 2\sqrt{mk}$ (т.е. $\gamma = \omega_0$): критическое демпфирование.

При $\beta > 2\sqrt{mk}$ (т.е. $\gamma > \omega_0$): апериодическое движение.

**Теорема 7.2 (Вынужденные колебания при гармоническом воздействии).** Для $F(t) = F_0\cos(\Omega t)$ установившееся решение:
$$x_{\text{уст}}(t) = A(\Omega)\cos(\Omega t - \varphi(\Omega))$$
где амплитуда:
$$A(\Omega) = \frac{F_0}{\sqrt{(k - m\Omega^2)^2 + (\beta\Omega)^2}}$$
и фазовый сдвиг:
$$\tan\varphi(\Omega) = \frac{\beta\Omega}{k - m\Omega^2}$$

## 8. Понижение порядка

**Теорема 8.1 (Случай I: уравнение не содержит $y$).** Для уравнения $F(x, y', y'', \ldots, y^{(n)}) = 0$ подстановка $z = y'$ понижает порядок на 1.

**Теорема 8.2 (Случай II: уравнение не содержит $x$ явно).** Для уравнения $F(y, y', y'', \ldots, y^{(n)}) = 0$ подстановка $p = y'$ (рассматривая $p$ как функцию от $y$) с использованием $y'' = p\tfrac{dp}{dy}$ понижает порядок.

**Теорема 8.3 (Случай III: известно одно решение).** Если $y_1(x)$ — решение ЛОДУ второго порядка $y'' + p(x)y' + q(x)y = 0$, то второе решение:
$$y_2(x) = y_1(x)\int \frac{e^{-\int p(x)dx}}{y_1^2(x)}dx$$

## 9. Системы дифференциальных уравнений

**Определение 9.1.** Нормальная система ДУ первого порядка:
$$\frac{d\mathbf{y}}{dx} = \mathbf{f}(x, \mathbf{y})$$

**Определение 9.2.** Линейная система ДУ:
$$\frac{d\mathbf{y}}{dx} = A(x)\mathbf{y} + \mathbf{f}(x)$$

**Определение 9.3.** Линейная однородная система: $\mathbf{f}(x) \equiv 0$. Линейная система с постоянными коэффициентами: $A(x) = A = \text{const}$.

**Теорема 9.1 (Метод собственных векторов).** Для системы $\mathbf{y}' = A\mathbf{y}$ с постоянной матрицей $A$ решения имеют вид $\mathbf{y} = e^{\lambda x}\mathbf{v}$, где $\lambda$ — собственное значение и $\mathbf{v}$ — соответствующий собственный вектор матрицы $A$.

**Теорема 9.2 (Различные вещественные собственные значения).** Если $\lambda_1, \ldots, \lambda_n$ — различные вещественные собственные значения с собственными векторами $\mathbf{v}_1, \ldots, \mathbf{v}_n$, то общее решение:
$$\mathbf{y}(x) = C_1e^{\lambda_1 x}\mathbf{v}_1 + \cdots + C_ne^{\lambda_n x}\mathbf{v}_n$$

**Теорема 9.3 (Комплексные собственные значения).** Если $\lambda = a + ib$ — комплексное собственное значение, то соответствующие вещественные решения образуются из действительной и мнимой частей комплексного решения $e^{\lambda x}\mathbf{v}$.

**Определение 9.4.** Матричная экспонента:
$$e^{Ax} = \sum_{k=0}^{\infty}\frac{(Ax)^k}{k!}$$

**Теорема 9.4 (Решение через матричную экспоненту).** Решение системы $\mathbf{y}' = A\mathbf{y}$ с начальным условием $\mathbf{y}(0) = \mathbf{y}_0$:
$$\mathbf{y}(x) = e^{Ax}\mathbf{y}_0$$

## 10. Интегральные уравнения

**Определение 10.1.** Интегральное уравнение — уравнение, в котором неизвестная функция находится под знаком интеграла.

**Определение 10.2.** Ядро интегрального уравнения — функция $K(x, t)$ в интеграле.

**Определение 10.3.** Уравнение Фредгольма I рода: $\int_a^b K(x,t)\varphi(t)dt = f(x)$.

Уравнение Фредгольма II рода: $\varphi(x) - \lambda\int_a^b K(x,t)\varphi(t)dt = f(x)$.

**Определение 10.4.** Уравнение Вольтерра I рода: $\int_a^x K(x,t)\varphi(t)dt = f(x)$.

Уравнение Вольтерра II рода: $\varphi(x) - \lambda\int_a^x K(x,t)\varphi(t)dt = f(x)$.

**Определение 10.5.** Однородное интегральное уравнение: $f(x) \equiv 0$.

**Теорема 10.1 (Эквивалентность задачи Коши и интегрального уравнения).** Задача Коши $y' = f(x,y)$, $y(x_0) = y_0$ эквивалентна интегральному уравнению Вольтерра II рода:
$$y(x) = y_0 + \int_{x_0}^x f(t, y(t))dt$$

**Теорема 10.2 (Метод последовательных приближений Пикара).** Для уравнения Вольтерра II рода решение находится как предел последовательности:
$$y_0(x) = f(x), \quad y_{n+1}(x) = f(x) + \lambda\int_a^x K(x,t)y_n(t)dt$$
при подходящих условиях на $K$ и $f$.

## 11. Краевые задачи

**Определение 11.1.** Краевая задача — задача нахождения решения ДУ, удовлетворяющего условиям на границах интервала (краевым условиям).

**Определение 11.2.** Краевые условия первого рода (Дирихле): $y(a) = A$, $y(b) = B$.

**Определение 11.3.** Краевые условия второго рода (Неймана): $y'(a) = A$, $y'(b) = B$.

**Определение 11.4.** Краевые условия третьего рода (Робина): $\alpha_1 y(a) + \beta_1 y'(a) = A$, $\alpha_2 y(b) + \beta_2 y'(b) = B$.

**Определение 11.5.** Задача Штурма-Лиувилля — краевая задача вида:
$$\frac{d}{dx}\left(p(x)\frac{dy}{dx}\right) - q(x)y + \lambda r(x)y = 0$$
с однородными краевыми условиями, где $p(x) > 0$, $q(x) \geq 0$, $r(x) > 0$.

**Определение 11.6.** Собственное значение $\lambda_n$ задачи Штурма-Лиувилля — значение параметра, при котором задача имеет нетривиальное решение.

**Определение 11.7.** Собственная функция $y_n(x)$ — соответствующее нетривиальное решение.

**Теорема 11.1 (О собственных значениях и функциях).**
1. Существует счётное множество собственных значений $\lambda_1 < \lambda_2 < \cdots \to \infty$
2. Каждому собственному значению соответствует собственная функция
3. Собственные функции образуют ортогональную систему (с весом $r(x)$) в $L^2[a, b]$

## 12. Семейства кривых и дифференциальные уравнения

**Теорема 12.1 (От семейства к ДУ).** Для семейства кривых $\Phi(x, y, C) = 0$ дифференцирование и исключение параметра $C$ дают ДУ, описывающее это семейство.

**Определение 12.1.** Ортогональные траектории семейства — кривые, пересекающие семейство под прямым углом в каждой точке пересечения.

**Теорема 12.2 (Ортогональные траектории).** Если семейство описывается ДУ $y' = f(x,y)$, то ортогональные траектории удовлетворяют:
$$y' = -\frac{1}{f(x,y)}$$

## 13. Особые решения

**Определение 13.1.** Особое решение — решение ДУ, в каждой точке которого нарушается условие единственности.

**Определение 13.2.** Огибающая семейства интегральных кривых $\Phi(x, y, C) = 0$ — кривая, касающаяся каждой интегральной кривой и являющаяся особым решением.

**Теорема 13.1 (Поиск огибающей).** Огибающая находится из системы:
$$\begin{cases}
\Phi(x, y, C) = 0 \\
\dfrac{\partial\Phi}{\partial C}(x, y, C) = 0
\end{cases}$$
путём исключения параметра $C$.

## 14. Уравнение Риккати

**Определение 14.1.** Уравнение Риккати:
$$\frac{dy}{dx} = a(x)y^2 + b(x)y + c(x)$$
где $a(x) \not\equiv 0$.

**Теорема 14.1 (Преобразование к линейному уравнению).** Если известно частное решение $y_0(x)$, то подстановка $y = y_0 + \dfrac{1}{z}$ преобразует уравнение Риккати в линейное уравнение первого порядка для $z$.

**Теорема 14.2 (Связь с ДУ второго порядка).** Уравнение Риккати подстановкой $y = -\dfrac{1}{a(x)}\dfrac{d}{dx}\ln u$ преобразуется в линейное ДУ второго порядка:
$$\frac{d^2u}{dx^2} + b(x)\frac{du}{dx} + a(x)c(x)u = 0$$

## 15. Уравнения Лагранжа и Клеро

**Определение 15.1.** Уравнение Лагранжа:
$$y = x\varphi(y') + \psi(y')$$
где $\varphi(p)$, $\psi(p)$ — известные функции, $p = y'$.

**Теорема 15.1 (Решение уравнения Лагранжа).** Дифференцированием и преобразованием уравнение сводится к линейному ДУ первого порядка относительно $x$ как функции от $p = y'$.

**Определение 15.2.** Уравнение Клеро (частный случай Лагранжа при $\varphi(p) = p$):
$$y = xy' + \psi(y')$$

**Теорема 15.2 (Решение уравнения Клеро).** Общее решение — семейство прямых:
$$y = Cx + \psi(C)$$

Особое решение (огибающая) — параметрически:
$$x = -\psi'(p), \quad y = -p\psi'(p) + \psi(p)$$

## 16. Приближённые методы

**Определение 16.1.** Метод ломаных Эйлера — численный метод решения задачи Коши $y' = f(x,y)$, $y(x_0) = y_0$ с рекуррентной формулой:
$$y_{i+1} = y_i + h \cdot f(x_i, y_i), \quad h = \frac{x_n - x_0}{n}$$

**Теорема 16.1 (Погрешность метода Эйлера).** Локальная ошибка метода Эйлера: $O(h^2)$. Глобальная ошибка: $O(h)$.

**Теорема 16.2 (Метод Эйлера-Коши).** Рекуррентная формула:
$$y_{i+1} = y_i + \frac{h}{2}[f(x_i, y_i) + f(x_{i+1}, y_i + hf(x_i, y_i))]$$
Глобальная ошибка: $O(h^2)$.

**Теорема 16.3 (Метод Рунге-Кутты четвёртого порядка).** Рекуррентная формула:
$$y_{i+1} = y_i + \frac{h}{6}(k_1 + 2k_2 + 2k_3 + k_4)$$
где $k_1 = f(x_i, y_i)$, $k_2 = f(x_i + \tfrac{h}{2}, y_i + \tfrac{h}{2}k_1)$, $k_3 = f(x_i + \tfrac{h}{2}, y_i + \tfrac{h}{2}k_2)$, $k_4 = f(x_i + h, y_i + hk_3)$.

Глобальная ошибка: $O(h^4)$.

## 17. Вспомогательные результаты

**Теорема 17.1 (Лемма Гронуолла).** Если $\varepsilon(x) \geq 0$ и $\varepsilon(x) \leq C + \int_{x_0}^x L \cdot \varepsilon(t)dt$ (где $C \geq 0$, $L > 0$), то:
$$\varepsilon(x) \leq Ce^{L(x-x_0)}$$

**Теорема 17.2 (О структуре пространства решений ЛОДУ).** Множество решений ЛОДУ $n$-го порядка образует линейное векторное пространство размерности $n$.

**Теорема 17.3 (О методе вариации постоянных).** Для ЛНДУ с известной ФСР однородного уравнения частное решение ищется в виде $y_{\text{ч.р.}} = C_1(x)y_1(x) + \cdots + C_n(x)y_n(x)$, где функции $C_k(x)$ определяются из системы линейных уравнений.

**Определение 17.1.** Функция Коши (ядро Коши) $G(x, s)$ для ДУ второго порядка — функция, удовлетворяющая однородному уравнению по переменной $x$, такая что $G(s, s) = 0$ и $\tfrac{\partial G}{\partial x}\big|_{x=s} = \tfrac{1}{a_0(s)}$.

**Теорема 17.4 (Применение функции Коши).** Решение ЛНДУ второго порядка с начальными условиями:
$$y(x) = y_0(x) + \int_{x_0}^x G(x, s)f(s)ds$$
где $y_0(x)$ — решение однородного уравнения с заданными начальными условиями.
