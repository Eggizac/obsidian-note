#  Что это
Словарь в Python это изменяемый тип данных, который представляет собой пару `ключ: значение`.
Создаются словари с помощью такого синтаксиса
``` python
dict = {

}
```

# Приемы со словарями

## Создание вложенных словарей

### Самое базовое
Рассмотрим это на примере продуктов для холодильника
``` python
products = {
	'bananas': {
		'date': '21.03.2021',
		'count': 21
	}
}
```
Это самое базовое создание

### С помощью функции
Допустим, что у нас уже создан словарь, тогда можно создать [[Функции в Python |функцию]] для этого
``` python
def add_product (category: str, data: str, count: int):
	products[category] = {
		'date': data,
		'count': count
	}
```
Таким образом, у нас появится новая категория в продуктах

## Получение доступа к данным из словарей

Получить все, что находится в слова можно с помощью метода `items()`

Если же у нас вложенный словарь, то можно воспользоваться следующей конструкцией
``` python
for key, value in dictionary['sub_name'].items():
	print(key, value)
```
