# Вводная информация
В этой части будет рассказано, что такое sql, структура запроса sql, стиль оформления sql запроса

## Что такое sql

**SQL** - это язык запросов для получения данных из баз данных.  

## Структура запроса SQL

1. **Ключевые слова** - это слова которые определены в SQL, например, SELECT, COUNT, ASC, KEY
2. **Интедефикаторы** - это имена объектов базы данных, например, имена таблиц в бд
3. **Оператор** - это символ или слово, указывающий на действие, выполняемое над одним или несколькими выражениями

## Стиль оформления SQL запроса

1. Все ключевые слова пишутся в верхнем регистре 
2. Каждое ключевое слово пишется с новой строчки
