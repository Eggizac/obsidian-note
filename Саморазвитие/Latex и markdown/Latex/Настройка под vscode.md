## Используемые плагины для LaTeX в VS Code

- **LaTeX Workshop**  
  Основное расширение для работы с LaTeX: сборка, предпросмотр PDF, подсветка, работа с математикой, структура документа и панель Math Preview.

- **HyperSnips (+ HyperScopes)**  
  Позволяет создавать расширенные сниппеты с автозаменами целых строк, что сильно ускоряет набор формул и шаблонов.

- **LaTeX Snippets**  
  Инструмент для удобного рисования графиков и генерации кода TikZ/PGFPlots.

- **Paste Image**  
  Быстро вставляет скриншоты и изображения в LaTeX-документ с автоматическим сохранением и генерацией команды \includegraphics.

- **Punto Switcher** (внешняя программа для Windows)  
  Автоматически переключает раскладку клавиатуры, облегчая ввод текста и LaTeX-команд.

## Настройка сниппетов

1. **LaTeX Workshop (latex.json)**  
   - Создаётся через меню настроек VS Code (Configure User Snippets).  
   - Используется для подсказок автодополнения, активируемых Enter или Tab.  
   - Пример: триггер `\lim` разворачивается в полный шаблон с пределами и индексами.

2. **HyperSnips (latex.hsnips, markdown.hsnips)**  
   - Файлы с автозаменами для целых строк, что позволяет, например, заменять символ `/` на конструкцию `\dfrac{}{}` с навигацией по табстопам.  
   - В сниппетах внутри скобок ставятся пробелы для удобного движения курсора по формуле.

3. **Общая идея**  
   - Комбинация двух типов сниппетов обеспечивает удобство и скорость ввода как простых, так и сложных конструкций.

## Набор математики и рисунков

- Использование панели Math Preview из LaTeX Workshop для отдельного отображения формул в процессе набора.
- Возможность вести конспекты в Markdown с поддержкой отображения формул.
- Для создания рисунков используется mspaint или аналогичный лёгкий редактор с ручным сохранением и вставкой.
- LaTeX Snippets помогают создавать код для графиков на TikZ/PGFPlots.
- Расширение Paste Image облегчает вставку скриншотов в документ.

---

## Работа с раскладкой клавиатуры

- Автоматизация переключения раскладки с помощью Punto Switcher на Windows.  
- Настройка горячих клавиш для смены раскладки выделенного текста и отмены автозамены.  
- Отключение автозамены раскладки непосредственно в VS Code для предотвращения ложных замен.
- Автор предлагает автоматизировать переключение раскладки при входе/выходе из математических блоков с помощью скриптов и горячих клавиш.
