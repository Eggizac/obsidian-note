#Английский #Английский_грамматика #Предложения

## Формула построения предложения

| Формула                                                                                     | Пример                                            |
| ------------------------------------------------------------------------------------------- | ------------------------------------------------- |
| Подлежащие + сказуемое                                                                      | I work, I see, You understand, We work, They work |
| Подлежащие + сказуемое + остальное                                                          | I like it                                         |
| Подлежащее + сказуемое + and + сказуемо                                                     | I work and travel                                 |
| I like (мне нравиться) + остальное                                                          | I like music                                      |
| I + usually (и подобное) + глагол                                                           | I usually do it in the morning, I often go there  |
| Very well в конце предложения                                                               | You speak English very well                       |
| Every day/Every year в конце предложения                                                    | I go there every year                             |
| I + also + глагол                                                                           | I also think so                                   |
| I/We/You/They + like ...                                                                    | We like this language                             |
| Если что-то действительно нравиться. I like + глагол + ing, I like + to + глагол (привычка) | I like traveling, I like to traver                |
| He/She/it + глагол + s                                                                      | He works                                          |
| He/She/it + глагол + s + ...                                                                |                                                   |
