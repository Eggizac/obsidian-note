# Используется
| Когда используется                                              | Пример                                                                          |
| --------------------------------------------------------------- | ------------------------------------------------------------------------------- |
| Действие началось в прошлом и продолжается сейчас               | I have been learning english for a six mounts - Я учил английский 6 месяцев<br> |
| Действие, которое только что закончилось и виден результат      | You have a red eye. Have you been crying - У тебя красные глаза. ты плакал?     |
| Повторяющиеся действия в последнее время (часто с раздражением) | He has calling me every hour! - Он звонит мне каждый час!                       |
# Формулы
| Какая форма | Формула                                       | Пример                                                |
| ----------- | --------------------------------------------- | ----------------------------------------------------- |
| Утверждение | Подлежащее + have/has + been + $V_{ing}$      | I have been studying english for a two hour every day |
| Вопрос      | Have/has + подлежащее + been + $V_{ing}$      | Have you been waiting long?                           |
| Отрицание   | Подлежащее + have/has+ not + been + $V_{ing}$ | They haven't been sleeping well lately                |
# Слова маркеры
