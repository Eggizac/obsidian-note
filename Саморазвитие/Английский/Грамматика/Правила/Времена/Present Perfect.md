# Основная информация
Это время связывает прошедшее и настоящее. 

Если я могу ответить на вопрос "Почему это важно сейчас?", то это Present perfect.

Время очень похоже на Past simple. Оно отличается от него тем, что результат важен сейчас, а в past simple результат не важен.

# Когда используется
| Случай                                                         | Пример                                                     |
| -------------------------------------------------------------- | ---------------------------------------------------------- |
| Опыт (делали что-то когда либо, и знаете то о чем говорите)    | I have read this book (я знаю то, о чем говорится в книге) |
| Недавно завершенное действие (только что)                      | I've just installed Python                                 |
| Действие, которое началось в прошлом и продолжается до сих пор | I have worked this coffee shop for two months              |
| Только что полученная новость/изменение                        | The program has crashed                                    |
# Формула
| Случай      | Формула                               |
| ----------- | ------------------------------------- |
| Утверждение | have/has + $V_3$                      |
| Отрицание   | have/has not (haven't/hasn't) + $V_3$ |
| Вопрос      | Have/has + подлежащее + $V_3$         |
