
**Используется:**
1. Цель предназначения - "Для чего? Для кого?"
	1. This documentation is for beginners - Эта документация для новичков
	2. I bought laptop for coding - Я купил ноутбук для программирования
	3. This function is for calculating the sum - Эта функция для вычисления суммы
2. Продолжительность времени - "В течении какого времени?"
	1. I worked in a coffee shop for two hours - Я работал в кофейне 2 часа
3. Причина мотив (из-за, потому что)
	1. Thank you for your help - Спасибо за твою помощь
4. Поддержка, согласие
	1. I'm for open-source software - Я за открытый исходный код
5. Направление/получатель (для кого/кому предназначено)
	1. This coffee is special for you - Этот кофе специально для тебя
