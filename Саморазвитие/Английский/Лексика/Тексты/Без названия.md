# Внутрилабораторный контроль: Расчеты для ОК 1,509 мг/л

---

### 1. Расчет концентрации и погрешности методики

Разбавляем исходную пробу в десять раз.
$$X = \frac{1,509}{10} = 0,151$$
Находим концентрацию для измерения в кювете.

Считаем абсолютную погрешность по формуле стандарта.
$$\Delta = 0,01 + 0,11 \cdot 0,151 = 0,02661$$
Округляем полученное значение по правилам метрологии.
Округленное значение составит ровно **0,027 мг/л**.

Начальник требует перевести значение в проценты.
$$\delta = \frac{0,027}{0,151} \cdot 100\% = 17,88\%$$
Получаем итоговую относительную погрешность нашей методики.
Она равна **18%** для контрольной карты.

---

### 2. Расчет границ карты Шухарта

Определяем предел действия для контрольной карты.
Предел действия составит ровно **18%**.

Вычисляем предел предупреждения по правилам контроля.
$$K_{\text{пр}} = 18\% \cdot 0,7 = 12,6\%$$
Эти параметры формируют ваши границы контроля.

---

### 3. Оценка практического результата ВЛК

Находим среднее арифметическое двух параллельных измерений.
$$X_{\text{лаб}} = \frac{1,498 + 1,510}{2} = 1,504$$
Вычисляем процентное отклонение от аттестованного значения.
$$\text{Отклонение} = \frac{1,504 - 1,509}{1,509} \cdot 100\% = -0,33\%$$
Точка идеально попадает в контрольные границы.

---

### 4. Погрешность приготовления исходного образца (1,509 мг/л)

Считаем погрешность приготовления исходного стандартного образца.
$$\delta_{\text{ОК исх}} = \sqrt{1^2 + 0,12^2 + 1^2 + 0,12^2}$$
Возводим в квадрат каждую отдельную погрешность.
$$1 + 0,0144 + 1 + 0,0144 = 2,0288$$
Извлекаем корень для получения промежуточного результата.
$$\delta_{\text{ОК исх}} = \sqrt{2,0288} \approx 1,424\%$$

---

### 5. Погрешность процедуры разбавления колбами

Оцениваем суммарную погрешность самой процедуры разбавления.
$$\delta_{\text{разб}} = \sqrt{0,12^2 + 0,24^2}$$
Складываем квадраты погрешностей пипетки и колбы.
$$0,0144 + 0,0576 = 0,072$$
Находим корень для погрешности этапа разбавления.
$$\delta_{\text{разб}} = \sqrt{0,072} \approx 0,268\%$$

---

### 6. Итоговая погрешность раствора в кювете (0,151 мг/л)

Объединяем погрешности приготовления и процедуры разбавления.
$$\delta_{\text{ОК итог}} = \sqrt{2,0288 + 0,072}$$
Суммируем значения под одним общим корнем.
$$\delta_{\text{ОК итог}} = \sqrt{2,1008} \approx 1,449\%$$
Грамотно округляем итоговую относительную погрешность образца.
Итоговая погрешность образца составит **1,45%**.

Вычисляем абсолютную погрешность для разбавленного раствора.
$$\Delta_{\text{ОК итог}} = 0,151 \cdot \frac{1,45}{100} \approx 0,0022$$

---

### 7. Проверка образца по правилам РМГ 76

Проверяем пригодность образца для внутреннего контроля.
Делим погрешность нашей методики на три.
$$\frac{18\%}{3} = 6\%$$
Погрешность образца намного меньше допустимого предела.
$$1,45\% < 6\%$$
Ваша система контроля выстроена абсолютно безупречно.
