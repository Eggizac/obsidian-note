1. Mean squared error (MSE) - среднеквадратичная функция потерь:
$$
MSE(\vec{y}, \vec{y^{'}}) = \frac{1}{n} \sum_{i=1}^n(y_i - y_i^{'})^2
$$
$\vec{y}$ - Вектор реальных значений (реальные цены на квартиры)
$\vec{y^{'}}$ - Вектор предсказанных значений
$y_i$ - Реальное значение для i-ого элемента
$y_i^{'}$ - Предсказанное значение для i-ого элемента
$n$ - Размер выборки
