Анализ нужен для того, чтобы подготовить данные к дальнейшему использованию в обучении модели. При анализе производится выявление закономерностей, появляется понимание о том какие переменные лишние, какие стоит заменить и тп.

1. Загрузить данные
2. Удалить все категориальные переменные (переменные, которые могут принимать ограниченное количество параметров). Удалять их надо потому что они не влияют на анализ
3. С помощью метода `describe` посмотреть на общую картину
4. Поиск закономерностей с помощью графиков и других инструментов в **вещественных** переменных
5. Поиск закономерностей в **категориальных** переменных
6. Составление итогового дата сета
