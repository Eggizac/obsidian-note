
| Термин на русском                      | Термин на английском | Что он означает                                                                                              |
| -------------------------------------- | -------------------- | ------------------------------------------------------------------------------------------------------------ |
| Выборка                                | DataSet              | Таблица данных                                                                                               |
| Объекты                                | Observation          | наименьшие элементы нашей выборки (одна строчка таблички)                                                    |
| Признаки                               | Feature              | атрибуты (свойства) объектов                                                                                 |
| Матрица плана                          | Desing Matrix        | матрица, которая объединяет в себе какие-то определенные признаки                                            |
| Целевая переменная                     | Target               | Это правильный ответ, который мы хотим получить от машины на основе имеющихся данных                         |
| Предсказанная переменная (отображение) | Prediction           | Ответ, который выдает модель, но основе имеющихся данных                                                     |
| Модель                                 | Model                | это некоторая формула, которая на основе имеющихся данных дает отображение                                   |
| Функция потерь                         | Loss function        | [[Loss function \| Функция, которая показывает, как хорошо наша модель предсказывает определенное значение]] |
| Параметр                               | Parametr             | Параметр, который модель использует, чтобы корректировать отображение                                        |
| Гиперпараметр                          | Hyperparametr        | Это признак, на основании которого модель подбирает параметры                                                |
