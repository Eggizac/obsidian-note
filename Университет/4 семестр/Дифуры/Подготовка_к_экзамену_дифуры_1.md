## ЧАСТЬ 1. ФАЗОВЫЕ ПОРТРЕТЫ (§16, задачи 961–978)

### Теория (кратко)

Система $\dot{x} = ax + by$, $\dot{y} = cx + dy$ (или уравнение $y' = \frac{cx+dy}{ax+by}$).

**Алгоритм:**

1. Привести к виду $y' = Q/P$ → записать матрицу $A = \begin{pmatrix} a & b \\ c & d \end{pmatrix}$ (знаменатель → первая строка, числитель → вторая).
2. Решить характеристическое уравнение $\lambda^2 - (a+d)\lambda + (ad-bc) = 0$.
3. Классифицировать по $\lambda_{1,2}$:

| Корни | Тип | Устойчивость |
|---|---|---|
| Вещ., одного знака | Узел | Уст. если $\lambda<0$ |
| Вещ., разных знаков | Седло | Неустойчиво |
| $\lambda_1 = \lambda_2$, диаг. | Дикритический узел | — |
| $\lambda_1 = \lambda_2$, жорд. | Вырожденный узел | — |
| $\alpha \pm \beta i$, $\alpha \neq 0$ | Фокус | Уст. если $\alpha<0$ |
| $\pm \beta i$ | Центр | Устойчив |

4. Для узла/седла: найти собственные векторы → прямые, через которые проходят/касаются траектории.
5. Для фокуса: определить направление вращения — подставить конкретную точку.

### Задача 971. $\dot{x} = 3x, \quad \dot{y} = 2x + y$

**Ответ по Филиппову: Узел.**

**Решение.** Матрица $A = \begin{pmatrix} 3 & 0 \\ 2 & 1 \end{pmatrix}$.

Характеристическое уравнение: $(3-\lambda)(1-\lambda) = 0$, откуда $\lambda_1 = 1$, $\lambda_2 = 3$.

Оба положительны → **неустойчивый узел**.

Собственные векторы:
- $\lambda_1 = 1$: $(A - E)v = 0$ → $\begin{pmatrix} 2 & 0 \\ 2 & 0 \end{pmatrix}v = 0$ → $v^1 = \begin{pmatrix} 0 \\ 1 \end{pmatrix}$ (ось $Oy$).
- $\lambda_2 = 3$: $(A - 3E)v = 0$ → $\begin{pmatrix} 0 & 0 \\ 2 & -2 \end{pmatrix}v = 0$ → $v^2 = \begin{pmatrix} 1 \\ 1 \end{pmatrix}$ (прямая $y = x$).

Общее решение: $x = c_1 e^{3t}$, $y = c_1 e^{3t} + c_2 e^{t}$ (подставляем во второе).

Фазовый портрет: $|\lambda_1| < |\lambda_2|$, поэтому при $t \to -\infty$ доминирует $e^t$ → траектории касаются оси $Oy$ (направление $v^1$). При $t \to +\infty$ доминирует $e^{3t}$ → траектории уходят вдоль $y = x$. Стрелки от начала координат (неустойчивость).

### Задача 974. $\dot{x} = x, \quad \dot{y} = 2x - y$

**Ответ по Филиппову: Седло.**

**Решение.** Матрица $A = \begin{pmatrix} 1 & 0 \\ 2 & -1 \end{pmatrix}$.

Характеристическое уравнение: $(1-\lambda)(-1-\lambda) = 0$, откуда $\lambda_1 = 1$, $\lambda_2 = -1$.

Разных знаков → **седло** (всегда неустойчиво).

Собственные векторы:
- $\lambda_1 = 1$: $\begin{pmatrix} 0 & 0 \\ 2 & -2 \end{pmatrix}v = 0$ → $v^1 = \begin{pmatrix} 1 \\ 1 \end{pmatrix}$ (прямая $y = x$ — неустойчивая сепаратриса).
- $\lambda_2 = -1$: $\begin{pmatrix} 2 & 0 \\ 2 & 0 \end{pmatrix}v = 0$ → $v^2 = \begin{pmatrix} 0 \\ 1 \end{pmatrix}$ (ось $Oy$ — устойчивая сепаратриса).

Фазовый портрет: по прямой $y = x$ точки убегают от начала ($e^t$ растёт), по оси $Oy$ — приближаются к началу ($e^{-t}$ убывает). Остальные траектории — гиперболоподобные кривые.

### Задача 973. $\dot{x} = x + 3y, \quad \dot{y} = -6x - 5y$

**Ответ по Филиппову: Фокус.**

**Решение.** Матрица $A = \begin{pmatrix} 1 & 3 \\ -6 & -5 \end{pmatrix}$.

Характеристическое уравнение: $\lambda^2 + 4\lambda + 13 = 0$ → $D = 16 - 52 = -36$ → $\lambda_{1,2} = -2 \pm 3i$.

Комплексные с $\alpha = -2 < 0$ → **устойчивый фокус** (спирали к началу).

Направление вращения: в точке $(1, 0)$: $\dot{x} = 1$, $\dot{y} = -6$. Вектор $(1, -6)$ указывает вправо-вниз → движение по часовой стрелке.

---

### Задача 975. $\dot{x} = -2x - 5y, \quad \dot{y} = 2x + 2y$

**Ответ по Филиппову: Центр.**

**Решение.** Матрица $A = \begin{pmatrix} -2 & -5 \\ 2 & 2 \end{pmatrix}$.

Характеристическое уравнение: $\lambda^2 + 6 = 0$ → нет, проверим: $(−2−\lambda)(2−\lambda) + 10 = 0$ → $\lambda^2 - 4 + 10 = 0$... Пересчитаем: $\det(A - \lambda E) = (-2-\lambda)(2-\lambda) - (-5)(2) = -4+\lambda^2 + 10 = \lambda^2 + 6 = 0$.

$\lambda_{1,2} = \pm i\sqrt{6}$ — чисто мнимые → **центр**.

Траектории — замкнутые кривые (эллипсы). Находим первый интеграл: из системы получаем уравнение $\frac{dy}{dx} = \frac{2x+2y}{-2x-5y}$. Можно найти квадратичный первый интеграл $2x^2 + 2xy + \frac{5}{2}y^2 = C$ (проверяется дифференцированием вдоль системы).

Направление: в точке $(1, 0)$: $\dot{x} = -2$, $\dot{y} = 2$ → влево-вверх → против часовой стрелки.

### Задача 972. $\dot{x} = 2x - y, \quad \dot{y} = x$

**Ответ по Филиппову: Вырожденный узел.**

**Решение.** Матрица $A = \begin{pmatrix} 2 & -1 \\ 1 & 0 \end{pmatrix}$.

$\lambda^2 - 2\lambda + 1 = 0$ → $\lambda_1 = \lambda_2 = 1$ (кратный корень).

Проверим: $A - E = \begin{pmatrix} 1 & -1 \\ 1 & -1 \end{pmatrix}$ — ранг 1, значит один собственный вектор → жорданова клетка → **вырожденный узел** (неустойчивый, т.к. $\lambda = 1 > 0$).

Собственный вектор: $(A - E)v = 0$ → $v = \begin{pmatrix} 1 \\ 1 \end{pmatrix}$.

Общее решение: $\mathbf{x} = c_1 e^t \begin{pmatrix} 1 \\ 1 \end{pmatrix} + c_2 e^t \left[t\begin{pmatrix} 1 \\ 1 \end{pmatrix} + \begin{pmatrix} -1 \\ 0 \end{pmatrix}\right]$ (присоединённый вектор $w$: $(A-E)w = v$ → $w = \begin{pmatrix} -1 \\ 0 \end{pmatrix}$... Проверим: $\begin{pmatrix} 1 & -1 \\ 1 & -1 \end{pmatrix}\begin{pmatrix} -1 \\ 0 \end{pmatrix} = \begin{pmatrix} -1 \\ -1 \end{pmatrix}$. Нужно $(A-E)w = v = (1,1)^T$. Тогда $w_1 - w_2 = 1$, берём $w = (0, -1)^T$. Проверка: $(A-E)(0,-1)^T = (1,1)^T$ ✓).

Все траектории при $t \to \pm\infty$ касаются прямой $y = x$ (направление собственного вектора). Стрелки от начала.

### Задача 961. $y' = \frac{2x + y}{3x + 4y}$

**Ответ по Филиппову: Седло.**

**Решение.** Записываем как систему: $\dot{x} = 3x + 4y$, $\dot{y} = 2x + y$. Матрица $A = \begin{pmatrix} 3 & 4 \\ 2 & 1 \end{pmatrix}$.

$\lambda^2 - 4\lambda - 5 = 0$ → $\lambda_1 = 5$, $\lambda_2 = -1$.

Разных знаков → **седло**.

Собственные векторы:
- $\lambda = 5$: $\begin{pmatrix} -2 & 4 \\ 2 & -4 \end{pmatrix}v = 0$ → $v^1 = (2, 1)$. Прямая $y = x/2$ — неустойчивая сепаратриса.
- $\lambda = -1$: $\begin{pmatrix} 4 & 4 \\ 2 & 2 \end{pmatrix}v = 0$ → $v^2 = (1, -1)$. Прямая $y = -x$ — устойчивая сепаратриса.

### Задача 966. $y' = \frac{2x - y}{x - y}$

**Ответ по Филиппову: Центр.**

**Решение.** Система: $\dot{x} = x - y$, $\dot{y} = 2x - y$. Матрица $A = \begin{pmatrix} 1 & -1 \\ 2 & -1 \end{pmatrix}$.

$\lambda^2 + 1 = 0$ → $\lambda_{1,2} = \pm i$ — чисто мнимые → **центр**.

Траектории — эллипсы. Первый интеграл: $2x^2 - 2xy + y^2 = C$ (проверяется: $\frac{d}{dt}(2x^2 - 2xy + y^2) = 4x\dot{x} - 2\dot{x}y - 2x\dot{y} + 2y\dot{y} = 4x(x-y) - 2(x-y)y - 2x(2x-y) + 2y(2x-y) = 4x^2 - 4xy - 2xy + 2y^2 - 4x^2 + 2xy + 4xy - 2y^2 = 0$ ✓).

## ЧАСТЬ 2. УРАВНЕНИЯ В ЧАСТНЫХ ПРОИЗВОДНЫХ (§20, стр. 122–128)

### Теория (кратко)

**Задача:** решить $a_1 \frac{\partial z}{\partial x} + a_2 \frac{\partial z}{\partial y} = b$ (квазилинейное уравнение).

**Алгоритм:**

1. Записать характеристическую систему: $\frac{dx}{a_1} = \frac{dy}{a_2} = \frac{dz}{b}$.
2. Найти два независимых первых интеграла: $\varphi_1(x,y,z) = C_1$, $\varphi_2(x,y,z) = C_2$.
3. Общее решение: $F(\varphi_1, \varphi_2) = 0$, или если $z$ входит только в один интеграл: $\varphi_2 = f(\varphi_1)$.
4. Задача Коши: подставить начальные данные → определить $f$.

**Для однородного** ($b = 0$, два переменных): $\frac{dx}{a_1} = \frac{dy}{a_2}$ → один первый интеграл $\varphi(x,y) = C$ → общее решение $z = f(\varphi)$.

---

### Задача 1167. $y\frac{\partial z}{\partial x} - x\frac{\partial z}{\partial y} = 0$

**Ответ: $z = f(x^2 + y^2)$.**

**Решение.** Характеристическая система: $\frac{dx}{y} = \frac{dy}{-x}$.

Домножаем крест-накрест: $x\,dx + y\,dy = 0$ → $x^2 + y^2 = C$.

Общее решение: $z = f(x^2 + y^2)$.

**Проверка.** $z_x = f' \cdot 2x$, $z_y = f' \cdot 2y$. Подставляем: $y \cdot 2xf' + (-x) \cdot 2yf' = 0$ ✓.

---

### Задача 1171. $y\frac{\partial z}{\partial x} + x\frac{\partial z}{\partial y} = x - y$

**Ответ: $F(x^2 - y^2, \, x - y - z) = 0$.**

**Решение.** Характеристическая система: $\frac{dx}{y} = \frac{dy}{x} = \frac{dz}{x-y}$.

Из первых двух: $\frac{dx}{y} = \frac{dy}{x}$ → $x\,dx - y\,dy = 0$ → $x^2 - y^2 = C_1$.

Замечаем: $\frac{dx - dy}{y - x} = \frac{dz}{x - y}$, то есть $\frac{d(x-y)}{-(x-y)} = \frac{dz}{x-y}$, откуда $dz = -d(x-y)$, значит $z + (x - y) = C_2$, т.е. $x - y - z = C_2$. Нет, $dz = -d(x-y)$ → $z = -(x-y) + C_2$ → $z + x - y = C_2$... Проверим иначе.

Из $\frac{dx}{y} = \frac{dz}{x-y}$ и $\frac{dy}{x} = \frac{dz}{x-y}$:

$\frac{dx - dy}{y - x} = \frac{dz}{x - y}$ → $\frac{d(x-y)}{-(x-y)} = \frac{dz}{x-y}$ → $(x-y)\,dz = -(x-y)\,d(x-y)$... Делим на $(x-y) \neq 0$: $dz = -d(x-y)$ → $z = -(x-y) + C$ → $z + x - y = C_2$.

Нет, тогда ответ $F(x^2-y^2, x-y+z) = 0$... Но в книге ответ $F(x^2-y^2, x-y-z) = 0$. Пересчитаем: $dz = -d(x-y)$ → $dz + d(x-y) = 0$ → $z + x - y = C_2$, то есть $x - y + z = C_2$, или эквивалентно $x - y - z = -C_2 = \tilde{C}_2$.

Оба варианта эквивалентны (функция $F$ произвольна). **Общее решение: $F(x^2 - y^2, \, x - y - z) = 0$**, или $x - y - z = f(x^2 - y^2)$, т.е. $z = x - y - f(x^2 - y^2)$.

### Задача 1175. $x\frac{\partial z}{\partial x} + 2y\frac{\partial z}{\partial y} = x^2 y + z$

**Ответ: $F\!\left(\frac{x^2}{y}, \, xy - \frac{3z}{x}\right) = 0$.**

**Решение.** Характеристическая система: $\frac{dx}{x} = \frac{dy}{2y} = \frac{dz}{x^2y + z}$.

Из первых двух: $\frac{dx}{x} = \frac{dy}{2y}$ → $2\ln|x| = \ln|y| + C$ → $\frac{x^2}{y} = C_1$.

Из первого и третьего: $\frac{dz}{dx} = \frac{x^2y + z}{x}$ → $\frac{dz}{dx} - \frac{z}{x} = xy$. Это линейное ОДУ по $z(x)$. На характеристике $y = x^2/C_1$, подставляем: $\frac{dz}{dx} - \frac{z}{x} = x \cdot \frac{x^2}{C_1} = \frac{x^3}{C_1}$.

Интегрирующий множитель: $\mu = e^{-\int -dx/x} = 1/x$ (нет, $\mu = e^{-\int (-1/x)dx} = e^{\ln x} = x$... Для уравнения $z' + pz = q$ с $p = -1/x$: $\mu = e^{\int p\,dx} = e^{-\ln x} = 1/x$).

$\frac{d}{dx}(z/x) = x^2/C_1$ → $z/x = \frac{x^3}{3C_1} + C_2'$ → $z = \frac{x^4}{3C_1} + C_2' x$. Подставляем $C_1 = x^2/y$: $z = \frac{x^4 y}{3x^2} + C_2' x = \frac{x^2 y}{3} + C_2' x$.

Значит $\frac{z}{x} - \frac{x^2 y}{3x} = C_2'$, т.е. $\frac{z}{x} - \frac{xy}{3} = C_2$, или $xy - \frac{3z}{x} = -3C_2 = \tilde{C}_2$.

**Общее решение:** $xy - \frac{3z}{x} = f\!\left(\frac{x^2}{y}\right)$.

### Задача 1189 (задача Коши). $x\frac{\partial z}{\partial x} - y\frac{\partial z}{\partial y} = 0, \quad z = 2x \text{ при } y = 1$

**Ответ: $z = 2xy$.**

**Решение.**

Шаг 1. Характеристическая система: $\frac{dx}{x} = \frac{dy}{-y}$.

$\ln|x| + \ln|y| = C$ → $xy = C_1$. Общее решение: $z = f(xy)$.

Шаг 2. Задача Коши: при $y = 1$ имеем $z = 2x$. Подставляем: $2x = f(x \cdot 1) = f(x)$ → $f(t) = 2t$.

$z = 2xy$.

### Задача 1196. $x\frac{\partial z}{\partial x} + y\frac{\partial z}{\partial y} = z - xy; \quad x = 2, \, z = y^2 + 1$

**Ответ: $(x + 2y)^2 = 2x(z + xy)$.**

**Решение.**

Шаг 1. Характеристическая система: $\frac{dx}{x} = \frac{dy}{y} = \frac{dz}{z - xy}$.

Из первых двух: $y/x = C_1$.

Из первого и третьего: $\frac{dz}{dx} = \frac{z - xy}{x} = \frac{z}{x} - y$. На характеристике $y = C_1 x$:
$z' - z/x = -C_1 x$ → решаем: $z/x = -C_1 x/2 + C_2'$ → $z = -C_1 x^2/2 + C_2' x = -xy/2 + C_2' x$.

Второй первый интеграл: $\frac{z + xy/2}{x} = C_2$, т.е. $\frac{z}{x} + \frac{y}{2} = C_2$, или $\frac{2z + xy}{2x} = C_2$.

Общее решение: $\frac{2z + xy}{2x} = f\!\left(\frac{y}{x}\right)$.

Шаг 2. Задача Коши: при $x = 2$, $z = y^2 + 1$:
$\frac{2(y^2+1) + 2y}{4} = f(y/2)$ → $\frac{2y^2 + 2y + 2}{4} = f(y/2)$.

Пусть $t = y/2$, тогда $y = 2t$: $f(t) = \frac{8t^2 + 4t + 2}{4} = 2t^2 + t + \frac{1}{2}$.

Подставляем обратно $t = y/x$:
$\frac{2z + xy}{2x} = 2\frac{y^2}{x^2} + \frac{y}{x} + \frac{1}{2}$

$2z + xy = 4\frac{y^2}{x} + 2y + x$

$2xz + x^2 y = 4y^2 + 2xy + x^2$

$(x+2y)^2 = x^2 + 4xy + 4y^2 = 2xz + x^2 y - x^2 y + x^2 + 4xy + 4y^2$...

Упрощаем иначе: $2z + xy = \frac{4y^2}{x} + 2y + x$ → $2xz + x^2 y = 4y^2 + 2xy + x^2$ → $2x(z + xy) = (x + 2y)^2$.

**Ответ:** $(x + 2y)^2 = 2x(z + xy)$ ✓.

### Задача 1190. $\frac{\partial z}{\partial x} + (2e^x - y)\frac{\partial z}{\partial y} = 0; \quad z = y \text{ при } x = 0$

**Ответ: $z = ye^x - e^{2x} + 1$.**

**Решение.**

Шаг 1. Характеристическая система: $\frac{dx}{1} = \frac{dy}{2e^x - y}$.

$\frac{dy}{dx} + y = 2e^x$ — линейное ОДУ. $\mu = e^x$.

$\frac{d}{dx}(ye^x) = 2e^{2x}$ → $ye^x = e^{2x} + C$ → $y = e^x + Ce^{-x}$.

Первый интеграл: $ye^x - e^{2x} = C_1$. Так как $b = 0$: $z = f(ye^x - e^{2x})$.

Шаг 2. При $x = 0$: $z = y$ → $y = f(y \cdot 1 - 1) = f(y - 1)$. Пусть $t = y - 1$ → $f(t) = t + 1$.

$z = (ye^x - e^{2x}) + 1 = ye^x - e^{2x} + 1$.

## ШПАРГАЛКА — АЛГОРИТМЫ

### Фазовый портрет
1. Записать матрицу $A$ (из системы или из уравнения $y' = Q/P$).
2. $\lambda^2 - \mathrm{tr}(A)\lambda + \det(A) = 0$.
3. Тип: по таблице.
4. Собственные векторы → оси портрета (для узла: касание к «медленному» вектору).
5. Стрелки: подставить точку $(1,0)$ или $(0,1)$.

### Уравнение в ч.п. первого порядка
1. Записать $\frac{dx}{a_1} = \frac{dy}{a_2} = \frac{dz}{b}$.
2. Из пар дробей: найти 2 первых интеграла.
3. Общее решение: $F(\varphi_1, \varphi_2) = 0$.
4. Задача Коши: $x = u(t), y = v(t), z = w(t)$ подставить в $\varphi_i$ → исключить параметр → найти $f$.
